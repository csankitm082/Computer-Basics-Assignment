Answer 2: RAM (Random Access Memory) is a type of computer memory that temporarily stores data and programs currently being used, allowing for quick access and processing by the CPU.

Answer 3 : Data in a computer is stored in the hard drive (HDD/SSD), RAM, cache memory, and external storage devices.

Answer 5 : Output devices include monitors, printers, speakers, projectors, and headphones.

Answer 6 : The input device that allows a user to move the cursor or pointer on the screen is a mouse.

Answer 7 : Machine language (or machine code) is directly understood by the computer without a translation program.
